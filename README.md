# LLLM

## AI Infrastructure Project

This repository is the single source of truth for the private AI infrastructure project.

It will contain architecture, procurement, design decisions, diagrams, configuration standards, operational runbooks, knowledge references, asset registers, warranty records, and lifecycle documentation for the local AI lab.

## Guiding Principles

- No compromise on quality.
- No overkill without technical justification.
- Amazon.in first for physical procurement.
- One master procurement workbook.
- Every major decision must be traceable through an ADR.
- Every system is frozen before the next system is finalized.
- Everything important is version-controlled.
